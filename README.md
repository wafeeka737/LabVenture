# LabVenture

## Project Overview
LabVenture is an interactive educational website designed to make learning science engaging and accessible. The website includes interactive activities, visual demonstrations, and explanations to help students understand scientific concepts more easily.

## Project Contents
- **index.html** – Main webpage  
- **style.css** – Styling for the website  
- **script.js** – JavaScript for interactive features (if any)  
- **images/** – All images used in the website  
- **Reports/** – Project reports and documentation  

## How to View
- Open `index.html` in a web browser to view the project locally.
